# /lesson — Compound Engineering 교훈 캡처

방금 완료한 작업에서 **반복하지 말아야 할 것**을 `.omc/lessons/` 에 기록합니다.
전역 지침 파일(AGENTS.md)에 추가하지 않습니다. 작고 집중된 파일로 분리합니다.

---

## 언제 쓰나

- `/ship` 또는 `/retro` 완료 직후
- 리뷰에서 반복 지적을 받았을 때
- 삽질로 시간을 낭비했을 때
- "다음엔 처음부터 이렇게 했어야 했다"는 생각이 들 때

---

## 교훈 캡처 양식

아래 질문에 답하며 양식을 채웁니다.

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
LESSON CAPTURE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
제목     : _______________________________________________
태그     : _______________________________________________

증상 (무슨 일이 발생했나):
  _______________________________________________

원인 (왜 처음부터 제대로 못 했나):
  _______________________________________________

적용된 규칙 (다음엔 어떻게):
  _______________________________________________

검증 커맨드 (규칙이 지켜졌는지 확인하는 법):
  _______________________________________________
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

양식을 채운 후:

```bash
python3 scripts/omc_lesson.py add \
  --title "제목" \
  --symptom "증상" \
  --cause "원인" \
  --rule "규칙" \
  --verify "검증 커맨드" \
  --tags "태그1,태그2"
```

또는 대화형:
```bash
python3 scripts/omc_lesson.py add -i
```

---

## 조회 커맨드

```bash
# 목록
python3 scripts/omc_lesson.py list

# 태그 필터
python3 scripts/omc_lesson.py list --tag tdd

# 검색
python3 scripts/omc_lesson.py search "false positive"

# 상세 보기
python3 scripts/omc_lesson.py show 2026-05-15-xxx.md
```

## 규칙

- 한 파일 = 한 교훈. 여러 교훈은 여러 파일로.
- 짧게 (제목 + 규칙 한 줄이면 충분).
- 다음 세션 시작 시 `omc_context.py`가 **BM25 유사도** 기반으로 현재 작업과 관련된 교훈을 자동 주입합니다.
- 교훈이 많을수록 BM25 매칭 정확도가 올라갑니다 — 꾸준히 기록하세요.
