# /benchmark — OMC 세계 1등 비교 분석

대상 기능을 사용자가 지정한 세계 1등 제품과 비교해 갭, 차별화, 다음 액션을 정리합니다.

## 실행

1. 상태 확인:

```bash
python3 scripts/omc.py state status --target .
```

2. 아래 입력을 확정합니다:
- 분석 기능
- 비교 대상
- 비교 관점
- 우리 현재 수준 근거

비교 대상이 없으면 후보 3개 이하만 제안하고 사용자 선택 전 임의 확정하지 않습니다.

3. 출력:
- 검증 상태: 미검증 / 직접 검증됨 / 공식 출처 확인됨
- 갭 분석 최소 3개
- 차별화 포인트 1개
- 우선순위 TOP 1
- 다음 액션: 실제 제품 검증 / `/office-hours` / `/plan`

근거 수준이 낮으면 `직접 검증 필요`를 붙이고 자동 구현으로 넘어가지 않습니다.

## 다음 추천

- 주추천 1개만 제시: 근거가 약하면 실제 제품 검증, 방향 검증이 필요하면 `/office-hours`, 범위가 선명하면 `/plan`
- 자동으로 진행하지는 않습니다.
