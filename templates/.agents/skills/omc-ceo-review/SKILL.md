# OMC 기능 범위 재검토 (CEO 관점)

모드: `EXPAND` | `SELECTIVE` | `HOLD` (기본) | `REDUCE`

호출 예: `$omc-ceo-review REDUCE 로그인 기능`

```
[ ] 1. 이 기능이 없으면 사용자가 이탈하는가?
[ ] 2. 핵심 가치 제안을 강화하는가?
[ ] 3. 단순한 버전으로도 80%의 가치를 낼 수 있는가?
[ ] 4. 성공 지표 1개: _______________
[ ] 5. 2주 안에 배포 가능한가?

결론: APPROVED / REDUCE / REJECT
이유: _______________
```
