# P2: SSOT 자동화 + 교훈 콘솔 출력
날짜: 2026-05-18
태그: ssot,sync,lesson,context,p2

## 증상
매번 cp 명령으로 SSOT를 수동 동기화해야 했음. 교훈은 context.md에만 있어 AI가 능동적으로 읽어야 했음

## 원인
동기화 스크립트 없음 + omc_context.py가 교훈을 파일에만 저장하고 콘솔 출력 없음

## 적용된 규칙
python3 scripts/omc_sync_ssot.py 로 SSOT 동기화. omc_context.py 실행 시 관련 교훈 N개를 터미널에 직접 출력

## 검증 커맨드
python3 scripts/omc_sync_ssot.py --check && python3 scripts/omc_context.py --target . | grep 교훈
