# 스킬은 편의성, 강제 진입점은 CLI에 있어야 한다
날짜: 2026-05-25
태그: skill,cli,preflight,autopilot

## 증상
omc-autopilot 스킬만 만들면 CLI 직접 호출 시 pre-flight 체크가 우회됐다

## 원인
스킬은 AI 지시 문서일 뿐, 강제 실행 경로가 아님

## 적용된 규칙
안전 장치(빈값 차단·짧은 지시문 경고)는 CLI argparse에 두고, 스킬은 편의성·안내 레이어로 설계

## 검증 커맨드
pytest scripts/test_omc_autopilot_preflight.py -v
