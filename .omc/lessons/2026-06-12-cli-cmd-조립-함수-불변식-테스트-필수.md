# CLI cmd 조립 함수 불변식 테스트 필수
날짜: 2026-06-12
태그: cmd,cli,tdd,insert

## 증상
insert(-1) 두 번 호출로 --reasoning-effort 값이 prompt_text로 덮어써짐

## 원인
insert(-1) 패턴은 마지막 요소 앞 삽입이므로 두 번 호출하면 순서 반전

## 적용된 규칙
CLI cmd 조립 함수는 positional arg(prompt_text 등)가 마지막임을 검증하는 테스트 케이스를 반드시 포함해야 한다
